# Restify Skeleton Application for mysql

## Install dependencies

```bash
# with npm
npm i
```
## Config

In the .env file the access configuration of the mysql database.

```bash
MYSQL_HOST=
MYSQL_USERNAME=
MYSQL_PASSWORD=
MYSQL_DATABASE=
MYSQL_TEST_DATABASE=
```

## Criate routes

Route configuration must be done in `src/http/routes.js`

## Start dev service
```bash
npm run dev
```
